== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Plugin Settings and fill settings.
The weather data is provided for free by http://openweathermap.org
Once you have the <a href="http://openweathermap.org/appid#get" target="_blank">API Key</a> you can simply add it in 'Settings' -> 'Lonely Planet Weather' and you're ready to go.

Insert this shortcode to your post: [pl-weather location="Nashville, TN" units="F"]
